Celtics Player's Name Goes Here

Use internal (or external) JavaScript to implement the features below:

  1. Display image of the player corresponding to the button the user clicks on.
  2. Change the color of the image border based on the button displayed: yellow for Al, red for Jaylen, green for Jayson and lime for Payton. If my color choices are poor, you can pick your own colors as long as it is different and unique for each image.
  3. Change the heading, "Celtics Player's Name Goes Here" to the name of player shown.
  4. Change the background color of the button that the user clicks on to Silver. Only the button the user has clicked on must be Silver. All others must be White.
  5. Enlarge the image by 25% when the user mouses-over it and return it to normal size when the user mouses-out of the image.
  6. (Optional) Try Array and for, loop so only one function is needed for 1-4.
